## Prerequisites

Before you start, make sure you have the following installed:

- **Python 3.10, 3.11, or 3.12** — other versions will not work
- **Git** — to clone the repository
- **FFmpeg** — required for video assembly (the bot will attempt to install it automatically on supported systems)

---

## Installation

<Steps>
  <Step title="Clone the repository">
    <Tabs>
      <Tab title="macOS / Linux">
        ```bash
        git clone https://github.com/elebumm/RedditVideoMakerBot.git
        cd RedditVideoMakerBot
        ```
      </Tab>
      <Tab title="Windows">
        ```bat
        git clone https://github.com/elebumm/RedditVideoMakerBot.git
        cd RedditVideoMakerBot
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Create and activate a virtual environment">
    <Tabs>
      <Tab title="macOS / Linux">
        ```bash
        python3 -m venv ./venv
        source ./venv/bin/activate
        ```
      </Tab>
      <Tab title="Windows">
        ```bat
        python -m venv ./venv
        .\venv\Scripts\activate
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Install Python dependencies">
    ```bash
    pip install -r requirements.txt
    ```

    This installs all required packages, including `praw`, `moviepy`, `playwright`, `gTTS`, `rich`, `toml`, `spacy`, `torch`, `transformers`, `elevenlabs`, `yt-dlp`, and others listed in `requirements.txt`.
  </Step>

  <Step title="Install Playwright and its browser dependencies">
    ```bash
    python -m playwright install
    python -m playwright install-deps
    ```

    Playwright is used to take screenshots of Reddit posts and comments. The `install-deps` command installs the required system-level browser libraries.

    <Note>
      On some systems you may need to use `python3` instead of `python`. If you encounter errors, try `python3 -m playwright install`.
    </Note>
  </Step>
</Steps>

## One-line install script (experimental)

On macOS and Linux (Debian, Arch, Fedora, CentOS, and derivatives), you can run the automated install script instead of following the steps above:

```bash
bash <(curl -sL https://raw.githubusercontent.com/elebumm/RedditVideoMakerBot/master/install.sh)
```

This script installs system dependencies, downloads the bot, installs Python packages, and sets up Playwright. It can also be used to update an existing installation.

<Warning>
  This script is marked experimental. Review it before running. It requires `bash` and `curl`, and will prompt you before making changes unless you pass the `-y` flag.
</Warning>

The script supports optional flags:

| Flag | Effect |
|------|--------|
| `-y` | Assume yes to all prompts |
| `-d` | Install system dependencies only |
| `-p` | Install Python dependencies and Playwright only |
| `-b` | Download the bot only |
| `-l` | Download the bot and install Python dependencies |

## Set up a Reddit API app

The bot uses the Reddit API to fetch threads. You need to create a "script" type app on Reddit to get credentials.

<Steps>
  <Step title="Open the Reddit Apps page">
    Go to [reddit.com/prefs/apps](https://www.reddit.com/prefs/apps) and scroll to the bottom. Click **Create another app**.
  </Step>
  <Step title="Fill in the app form">
    - **Name**: any name, e.g. `VideoMakerBot`
    - **Type**: select **script**
    - **Redirect URI**: any valid URL, e.g. `https://localhost`

    Click **Create app**.
  </Step>
  <Step title="Copy your credentials">
    After the app is created, note down:
    - **Client ID** — shown under the app name (14-character string)
    - **Client Secret** — labeled "secret"

    You will enter these when the bot prompts you on first run.
  </Step>
</Steps>

## Run the bot

```bash
python main.py
```

On Windows, you can also use the provided batch script:

```bat
run.bat
```

### First-time configuration

On the first run, the bot checks for a `config.toml` file. If one does not exist, it walks you through an interactive setup prompt for each required value.

The bot will ask for the following in order:

**Reddit credentials** (`[reddit.creds]`):

| Prompt | Description |
|--------|-------------|
| `client_id` | Your Reddit app's client ID (12–30 characters) |
| `client_secret` | Your Reddit app's secret (20–40 characters) |
| `username` | Your Reddit account username |
| `password` | Your Reddit account password |
| `2fa` | Whether 2FA is enabled on your account (`true` / `false`) |

**Thread settings** (`[reddit.thread]`):

| Prompt | Description |
|--------|-------------|
| `subreddit` | Subreddit name to pull posts from, e.g. `AskReddit`. Separate multiple subreddits with `+`. |
| `random` | Whether to pick a random post (`true`) or use a specific post ID (`false`) |
| `post_id` | (Optional) Specific post ID to use |
| `max_comment_length` | Maximum characters per comment (default: `500`) |
| `min_comment_length` | Minimum characters per comment (default: `1`) |
| `min_comments` | Minimum number of comments a post must have (default: `20`) |

**General settings** (`[settings]`):

| Prompt | Description |
|--------|-------------|
| `theme` | Reddit UI theme: `dark`, `light`, or `transparent` |
| `times_to_run` | How many videos to generate in one run (default: `1`) |
| `allow_nsfw` | Whether to include NSFW posts (`true` / `false`) |
| `voice_choice` | TTS engine: `tiktok`, `elevenlabs`, `googletranslate`, `awspolly`, `streamlabspolly`, `pyttsx`, or `OpenAI` |

<Tip>
  To reconfigure any option after the initial setup, open `config.toml` and delete the lines you want to change. The bot will prompt you to re-enter those values on the next run.
</Tip>

---

## What happens when the bot runs

1. Fetches a thread from the chosen subreddit using the Reddit API
2. Generates TTS audio for the post title and each comment
3. Takes Playwright screenshots of the Reddit post and comments
4. Downloads the background video and audio tracks
5. Chops the background to match the video length
6. Assembles the final video using FFmpeg and saves it to the `results/` directory
