---
title: "TikTok TTS"
description: "Set up the TikTok Text-to-Speech engine, obtain your session ID from browser cookies, and choose from 40+ voices including Disney characters."
---

The TikTok TTS engine calls TikTok's internal speech API (`api16-normal-c-useast1a.tiktokv.com`) using your session cookie for authentication. No paid account or developer key is needed — a regular logged-in TikTok account is all that is required.

<Warning>
  Your `sessionid` cookie expires periodically. If TTS generation starts failing, revisit the steps below to grab a fresh session ID.
</Warning>

## Setup

<Steps>
  <Step title="Log in to TikTok in your browser">
    Open [tiktok.com](https://www.tiktok.com) and sign in to your account.
  </Step>
  <Step title="Open browser developer tools">
    Press `F12` (or `Cmd+Option+I` on macOS) to open DevTools, then navigate to the **Application** tab (Chrome/Edge) or **Storage** tab (Firefox).
  </Step>
  <Step title="Find the sessionid cookie">
    Under **Cookies → https://www.tiktok.com**, locate the cookie named `sessionid`. Copy its **Value**.
  </Step>
  <Step title="Add the session ID to your config">
    Paste the value into `.config.toml`:

    ```toml
    [settings.tts]
    voice_choice       = "tiktok"
    tiktok_sessionid   = "c76bcc3a7625abcc27b508c7db457ff1"
    tiktok_voice       = "en_us_001"
    random_voice       = false
    ```
  </Step>
</Steps>

## Config Reference

<ParamField path="voice_choice" type="string" required>
  Must be set to `"tiktok"` to activate this engine.
</ParamField>

<ParamField path="tiktok_sessionid" type="string" required>
  Your TikTok `sessionid` cookie value. The engine injects this as a `Cookie` header on every request.

  Example: `"c76bcc3a7625abcc27b508c7db457ff1"`
</ParamField>

<ParamField path="tiktok_voice" type="string" default="en_us_001">
  The voice code to use when `random_voice` is `false`. If this key is absent from the config, the API selects the most fitting voice automatically.

  Example: `"en_us_006"`
</ParamField>

<ParamField path="random_voice" type="bool" default="true">
  When `true`, a random English voice is chosen from `eng_voices` for each comment. Disney and vocal voices are excluded from random selection.
</ParamField>

<Note>
  The TikTok engine accepts a maximum of **200 characters** per API request. Longer comments are automatically split by `TTSEngine.split_post()` and re-joined with a silence gap.
</Note>

  The `random_voice` option only randomises across the **English** voice set (`eng_voices`). To use a Disney or vocal voice, set `random_voice = false` and specify the voice code explicitly in `tiktok_voice`.
</Tip>

## Error Codes

The TikTok API returns a `status_code` field in each response. The engine raises a `TikTokTTSException` on non-zero codes:

| Code | Reason |
|---|---|
| `1` | Incorrect `aid` value (internal API issue) |
| `2` | Text too long (over 200 characters) |
| `4` | Specified voice does not exist |
