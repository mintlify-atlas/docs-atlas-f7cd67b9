The bot supports multiple TTS engines. Set `voice_choice` in `[settings.tts]` to the value in the table below.

## TTS Engines

| Engine | `voice_choice` value | Free | Requires API Key |
|---|---|---|---|
| TikTok | `tiktok` | Yes | No (requires `tiktok_sessionid`) |
| Google Translate | `googletranslate` | Yes | No |
| pyttsx3 (system TTS) | `pyttsx` | Yes | No |
| Streamlabs Polly | `streamlabspolly` | Yes | No |
| AWS Polly | `awspolly` | Paid | Yes (AWS credentials) |
| ElevenLabs | `elevenlabs` | Paid | Yes (`elevenlabs_api_key`) |
| OpenAI | `OpenAI` | Paid | Yes (`openai_api_key`) |

<Note>

## AWS Polly

Set `voice_choice = "awspolly"`. The default voice is `Matthew`. Requires the AWS CLI installed and a named profile called `polly` configured via `aws configure --profile polly`.

```toml
[settings.tts]
voice_choice = "awspolly"
aws_polly_voice = "Matthew"
```

---

## Streamlabs Polly

Set `voice_choice = "streamlabspolly"`. The default voice is `Matthew`. No API key is required.

```toml
[settings.tts]
voice_choice = "streamlabspolly"
streamlabs_polly_voice = "Matthew"
```

## System TTS (pyttsx3)

Set `voice_choice = "pyttsx"`. This uses voices installed on your operating system. Windows includes 2 voices by default; additional voices can be installed.

Run `ptt.py` in the project root to list available voice indices on your system:

```bash
python ptt.py
```

Then set `python_voice` to the desired index and `py_voice_num` to the total number of voices installed.

```toml
[settings.tts]
voice_choice = "pyttsx"
python_voice = "0"
py_voice_num = "2"
```
